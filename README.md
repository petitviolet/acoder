## Structure

- ./api: API server
    - Ruby, Ruby on Rails
- ./frontend: Frontend SPA
    - TypeScript, React.js, Webpack

## Getting Started

### API

```console
$ cd api
$ docker-compose up -d
$ bundle
$ bundle exec rails s
```

### Frontend

```console
$ cd frontend
$ yarn
$ yarn start
```
